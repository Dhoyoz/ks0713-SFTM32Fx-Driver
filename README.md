# ks0713-SFTM32Fx-Driver SPI
Conections:

LCD-------------STM32Fx

CS1B -------->  LCD_SPI_CS

LCD_RS ------>  RS

LCD_RST ----->  RST

DB6  -------->  SPI1_SCK

DB7  -------->  SPI1_MOSI

Remenber closed J3 Jumper on back LCD to activate serial comunication
